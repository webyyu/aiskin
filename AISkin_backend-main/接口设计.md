在21天护肤计划（TwentyOneDayPlan.vue）中，上传照片进行AI检测用的接口是：

**POST** `/api/skin-analysis/analyze`

- **请求方式**：POST
- **请求头**：`Authorization: Bearer <token>`（用户登录后的token）
- **请求体**：`multipart/form-data`，字段名为`faceImage`，即上传的面部照片
- **返回**：AI分析后的肤质数据（如水分、光泽度、弹性、问题区域评分、皮肤类型、整体评估等）

**完整调用示例：**
```js
const formData = new FormData();
formData.append('faceImage', file); // file为用户选择的图片文件

fetch('http://localhost:5000/api/skin-analysis/analyze', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + token // 用户token
  },
  body: formData
})
```

**总结：**
- 你只需要把用户当天的面部照片通过上述接口上传，后端会自动分析并返回结果，前端再展示分析数据即可。
- 该接口在21天护肤计划和其他需要AI肤质分析的地方都通用。
